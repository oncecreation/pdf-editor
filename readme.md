## How to use pdf-editor?

1. Click `Choose PDF` and choose a `.pdf` file.
2. Add images, drawing, text to your PDF.
3. Click `Save`.
4. That's it! All is done in your browser.

## Features

- Resize or move everything.
- Crop images.
